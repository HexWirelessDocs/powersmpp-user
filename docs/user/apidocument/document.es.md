
### Documentos de API: Acceso y Generar Solicitar URLs

El **API Documents** característica en iTextPro permite a los usuarios integrar perfectamente varias API, mejorando la funcionalidad de la plataforma. 
Los usuarios pueden acceder a la documentación completa de la API que contiene detalles esenciales como URLs de solicitud, estructuras de carga y directrices de uso.

![API Documents](images/document1.png)

---

#### Características clave

- **API Integration Opportunities** 
 Explore múltiples APIs para ampliar y personalizar las capacidades de iTextPro.

- **Documentación completa de API** 
 Guías claras, concisas y completas con toda la información necesaria para una integración fluida.

- **Solicitar generación de URL** 
 Usar el **Ver documento API** opción para generar y revisar diferentes URL de solicitud utilizadas dentro de iTextPro.

---

#### Pasos para acceder a documentos de API " Solicitar URLs

1. **Access API Documents** 
 Navigate a la **API Documents** sección dentro de la plataforma iTextPro.

2. **Ver documento API** 
 Haga clic **Ver documento API** abrir la referencia detallada de API.

3. **Generar URLs de solicitud** 
 Dentro del documento, generar y recuperar varias URL de solicitud para uso en integración.

---

El **API Documentos " Solicitar URLs** característica equipa a los usuarios con la información y herramientas necesarias para la integración eficaz de API. 
Desde explorar diversas opciones de API para generar URLs de solicitud, esta funcionalidad admite una experiencia de integración perfecta y eficiente.
