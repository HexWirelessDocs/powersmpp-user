
### API do desenvolvedor: Capacitando Integração com API SMS

A **API do desenvolvedor** o recurso no iTextPro permite que os usuários integrem várias APIs SMS, permitindo a submissão perfeita de mensagens SMS para o aplicativo. 
Esta secção fornece acesso a um **Documento da API**—um manual de referência completo que detalha as cargas úteis, os tipos de resposta e as melhores práticas para uma integração eficaz.

![Developer API](images/developerapi1.png)

---

#### Características Principais

- **Integração de API SMS** 
 Utilize vários terminais de API SMS para enviar SMS para o aplicativo iTextPro.

- **Documento de API abrangente** 
 Um guia claro, conciso e completo para desenvolvedores, cobrindo todos os detalhes de integração.

![API Document](images/developerapi2.png)

- **Manual de referência** 
 Inclui explicações detalhadas dos formatos de carga útil da solicitação, estruturas de resposta, autenticação e manipulação de erros.

- **Integração com a Ferramenta Swagger** 
 A **Ver documento da API** A opção abre a ferramenta Swagger — oferecendo uma interface de teste REST API segura, confiável e interativa para integração de terceiros.

---

#### Passos para acessar o API do desenvolvedor e o documento da API

![Access Developer API](images/developerapi3.png)

1. **Acessar a API do desenvolvedor** 
 Navegar para o **API do desenvolvedor** seção dentro da plataforma iTextPro.

2. **Ver documento da API** 
 Clique **Ver documento da API** para abrir a referência completa da API.

3. **Usar a Ferramenta do Swagger** 
 O sistema redireciona você para a ferramenta Swagger, onde você pode testar, explorar e integrar funcionalidades da API SMS.

---

A **API do desenvolvedor** no iTextPro capacita os desenvolvedores para integrar recursos SMS em suas próprias aplicações com facilidade. 
A combinação de uma referência detalhada da API com a interface de teste interativa do Swagger garante uma integração eficiente, segura e confiável.
