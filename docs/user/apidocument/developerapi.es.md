
### Developer API: Empowering Integration with SMS API

El **Developer API** característica en iTextPro permite a los usuarios integrar varias API de SMS, permitiendo la presentación perfecta de mensajes SMS a la aplicación. 
Esta sección proporciona acceso a una amplia **API Document**—un manual de referencia completo que detalla las cargas de pago, los tipos de respuesta y las mejores prácticas para una integración efectiva.

![Developer API](images/developerapi1.png)

---

#### Características clave

- **Integración de SMS API** 
 Utilizar varios puntos finales de API de SMS para enviar SMS a la aplicación iTextPro.

- **Archivo de API** 
 Una guía clara, concisa y completa para los desarrolladores, que abarca todos los detalles de integración.

![API Document](images/developerapi2.png)

- **Manual** 
 Incluye explicaciones detalladas de formatos de carga útil de solicitud, estructuras de respuesta, autenticación y manejo de errores.

- **Integración de herramientas de Swagger** 
 El **Ver documento API** La opción abre la herramienta Swagger —ofreciendo una interfaz de prueba REST API segura, fiable e interactiva para la integración de terceros.

---

#### Pasos para Access Developer API & API Document

![Access Developer API](images/developerapi3.png)

1. **Access Developer API** 
 Navigate a la **Developer API** sección dentro de la plataforma iTextPro.

2. **Ver documento API** 
 Haga clic **Ver documento API** abrir la referencia completa de API.

3. **Use Swagger Tool** 
 El sistema le redirige a la herramienta Swagger, donde puede probar, explorar e integrar funcionalidades de SMS API.

---

El **Developer API** en iTextPro faculta a los desarrolladores para integrar las capacidades de SMS en sus propias aplicaciones con facilidad. 
La combinación de una referencia de API detallada y la interfaz de prueba interactiva de Swagger garantiza una integración eficiente, segura y confiable.
