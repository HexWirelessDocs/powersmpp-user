
### API du développeur : renforcer l'intégration avec l'API SMS

Les **API développeur** fonctionnalité dans iTextPro permet aux utilisateurs d'intégrer diverses API SMS, permettant la soumission transparente de messages SMS à l'application. 
La présente section donne accès à une base de données **Document API**— un manuel de référence complet détaillant les charges utiles, les types de réponse et les meilleures pratiques pour une intégration efficace.

![Developer API](images/developerapi1.png)

---

#### Caractéristiques principales

- **Intégration de l'API SMS** 
 Utiliser plusieurs paramètres de l'API SMS pour soumettre des SMS à l'application iTextPro.

- **Document API complet** 
 Un guide clair, concis et complet pour les développeurs, couvrant tous les détails d'intégration.

![API Document](images/developerapi2.png)

- **Manuel de référence** 
 Comprend des explications détaillées sur les formats de charge utile de la demande, les structures de réponse, l'authentification et le traitement des erreurs.

- **Intégration de l'outil d'échange** 
 Les **Afficher le document de l'API** l'option ouvre l'outil Swagger, offrant une interface de test REST sécurisée, fiable et interactive pour l'intégration de tiers.

---

#### Étapes pour accéder au document API et API du développeur

![Access Developer API](images/developerapi3.png)

1. **Accès API développeur** 
 Naviguez vers la **API développeur** section dans la plate-forme iTextPro.

2. **Afficher le document de l'API** 
 Cliquez sur **Afficher le document de l'API** ouvrir la référence complète de l'API.

3. **Utiliser l'outil Swagger** 
 Le système vous redirige vers l'outil Swagger, où vous pouvez tester, explorer et intégrer les fonctionnalités de l'API SMS.

---

Les **API développeur** iTextPro permet aux développeurs d'intégrer facilement les capacités SMS dans leurs propres applications. 
La combinaison d'une référence API détaillée et d'une interface de test interactive de Swagger assure une intégration efficace, sécurisée et fiable.
