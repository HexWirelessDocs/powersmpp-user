
### Documentos API: Acesse e Gere URLs de Pedido

A **Documentos da API** O recurso no iTextPro permite que os usuários integrem perfeitamente várias APIs, melhorando a funcionalidade da plataforma. 
Os usuários podem acessar documentação abrangente da API contendo detalhes essenciais, como URLs de solicitação, estruturas de carga útil e diretrizes de uso.

![API Documents](images/document1.png)

---

#### Características Principais

- **Oportunidades de Integração da API** 
 Explore várias APIs para ampliar e personalizar os recursos do iTextPro.

- **Documentação abrangente da API** 
 Guias claros, concisos e completos com todas as informações necessárias para uma integração suave.

- **Pedir a Geração de URL** 
 Utilizar o **Ver documento da API** opção para gerar e rever diferentes URLs de solicitação usadas no iTextPro.

---

#### Passos para acessar documentos da API e solicitar URLs

1. **Acessar documentos da API** 
 Navegar para o **Documentos da API** seção dentro da plataforma iTextPro.

2. **Ver documento da API** 
 Clique **Ver documento da API** para abrir a referência detalhada da API.

3. **Gerar URLs de Pedido** 
 Dentro do documento, gerar e recuperar vários URLs de solicitação para uso na integração.

---

A **Documentos da API e URLs de solicitação** O recurso equipa os usuários com as informações e ferramentas necessárias para uma integração eficaz da API. 
Desde explorar diversas opções de API até gerar URLs de solicitação, esta funcionalidade suporta uma experiência de integração perfeita e eficiente.
