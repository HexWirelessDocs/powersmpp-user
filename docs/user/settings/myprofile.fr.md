
## Paramètres

Dans **Paramètres** section, les utilisateurs peuvent gérer les configurations de compte essentielles. Cette section contient les options suivantes:

- **Mon profil**
- **Modifier le mot de passe**

---

### Mon profil : Gérer les informations de votre compte

Les **Mon profil** option dans iTextPro permet aux utilisateurs de gérer leurs détails de compte de base, en assurant des informations précises et personnalisées.

![My Profile](images/profile1.png)

---

#### Caractéristiques principales

- **Prénom** 
 Entrez votre prénom, qui sera référencé par le compte parent lors de l'émission des factures.

- **Numéro mobile** 
 Inscrivez votre numéro mobile pour les communications et la vérification de compte.

- **Code pays / Ajout automatique Code pays** 
 Sélectionnez le code de votre pays à ajouter automatiquement aux numéros lors de l'envoi des campagnes. En option, activez la fonction d'ajout automatique pour plus de commodité.

- **Adresse électronique** 
 Fournissez votre adresse électronique pour la communication, les alertes, les courriels de bienvenue, les vérifications OTP et les notifications.

- **Pays/État** 
 Indiquez votre pays et votre État aux fins de facturation et de référence.

- **fuseau horaire** 
 Définissez votre fuseau horaire local pour vous assurer que toutes les données relatives à la date et à l'heure sont affichées avec précision selon votre emplacement.

---

#### Étapes pour gérer mon profil

1. **Accéder à mon profil** 
 Allez au **Mon profil** section dans le menu Paramètres.

2. **Mettre à jour les informations** 
 Remplissez ou mettez à jour des champs tels que prénom, numéro de téléphone mobile, code de pays, courriel, pays, état et fuseau horaire.

3. **Enregistrer les modifications** 
 Cliquez sur **Enregistrer** pour confirmer et appliquer vos mises à jour.

---

Les **Mon profil** fonctionnalité fournit aux utilisateurs un contrôle sur leurs détails de compte, permettant une expérience plus personnalisée et efficace.
