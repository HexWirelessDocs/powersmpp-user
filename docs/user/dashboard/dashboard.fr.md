
# Panneau utilisateur du Hub iText

## Module de l'utilisateur : Autonomiser les utilisateurs finaux
Le module utilisateur dans iTextPro est conçu pour responsabiliser les utilisateurs finaux avec un ensemble complet de fonctionnalités, leur permettant de gérer efficacement leurs campagnes SMS, contacts, trafic, rapports, profil et paramètres.

### Principales caractéristiques :
- **Nouveau déclencheur de campagne :** Lancez une nouvelle campagne SMS en toute transparence.
- **Gestion des contacts :** Gérer et organiser efficacement les contacts pour une communication ciblée.
- **Affichage du trafic :** Voir à la fois Mobile Originated (MO) et Mobile Termined (MT) trafic pour améliorer la surveillance.
- **Téléchargement du rapport :** Téléchargez des rapports détaillés pour des analyses et des idées approfondies.
- **Gestion du profil :** Gérer et mettre à jour facilement les profils des utilisateurs.
- **Gestion des paramètres :** Configurer et gérer différents paramètres pour personnaliser l'expérience utilisateur.

## Tableau de bord: Surveillance proactive et perspectives

![Dashboard](images/dashboard1.png)

Lors de la connexion au compte utilisateur, les utilisateurs sont accueillis avec le tableau de bord utilisateur, fournissant des informations précieuses sur les modèles de trafic SMS et les performances de la campagne.

### Aperçus disponibles sur le tableau de bord de l'utilisateur :
- **SMS envoyé aujourd'hui:** Affiche le nombre total de messages SMS envoyés par l'utilisateur à partir de différentes interfaces le jour actuel.
- **Livré aujourd'hui :** Souligne le nombre total de messages SMS livrés avec succès aux numéros de destination le jour actuel.
- **SMS envoyé hier:** Présente le nombre total de messages SMS envoyés par l'utilisateur la veille.
- **Livraison hier :** Affiche le nombre total de messages SMS livrés aux numéros de destination la veille.
