
# Painel de usuário iText Hub

## Módulo do usuário: Empoderando os usuários finais
O Módulo do Usuário no iTextPro é projetado para capacitar usuários finais com um conjunto abrangente de funcionalidades, permitindo que eles gerenciem eficientemente suas campanhas de SMS, contatos, tráfego, relatórios, perfil e configurações.

### Características chave:
- **Novo gatilho de campanha:** Iniciar uma nova campanha SMS sem problemas.
- **Gestão de Contactos:** Gerencie e organize efetivamente contatos para comunicação direcionada.
- **Visualização de Tráfego:** Veja o tráfego móvel originado (MO) e terminal (MT) para monitoramento aprimorado.
- **Transferência do Relatório:** Baixe relatórios detalhados para análises e insights detalhados.
- **Gerenciamento de perfil:** Gerencie e atualize facilmente os perfis de usuário.
- **Gestão de Configurações:** Configure e gerencie várias configurações para personalizar a experiência do usuário.

## Painel: Monitoramento e Insights Proativos

![Dashboard](images/dashboard1.png)

Ao iniciar sessão na conta de usuário, os usuários são recebidos com o Painel de Usuário, fornecendo informações valiosas sobre padrões de tráfego de SMS e desempenho de campanha.

### Insights Disponíveis no Painel do Usuário:
- **SMS enviado hoje:** Mostra o número total de mensagens SMS enviadas pelo usuário de diferentes interfaces no dia atual.
- **Entregue hoje:** Destaca o número total de mensagens SMS entregues com sucesso aos números de destino no dia atual.
- **SMS Enviado Ontem:** Apresenta a contagem total de mensagens SMS enviadas pelo usuário no dia anterior.
- **Entregue ontem:** Mostra o número total de mensagens SMS entregues aos números de destino no dia anterior.
