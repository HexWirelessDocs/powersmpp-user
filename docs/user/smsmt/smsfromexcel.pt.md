
# SMS do Excel: Acionando campanhas com arquivos do Excel

A **SMS do Excel** tab in *iTextPro* fornece aos usuários uma abordagem simplificada para desencadear campanhas usando um arquivo Excel contendo vários conteúdos de mensagens ou variáveis. Este recurso é particularmente útil para o envio de lembretes, ofertas promocionais personalizadas e muito mais.

![SMS From Excel](images/smsexcel1.png)

## Características Principais

### Envio do Ficheiro Excel
Os usuários podem desencadear campanhas de um arquivo Excel contendo números móveis e detalhes adicionais.

![Excel File Upload](images/smsexcel2.png)

### Download de arquivo de exemplo
Clique na **"Ver Amostra"** aba para baixar um arquivo Excel exemplo, fornecendo orientações sobre o formato necessário.

### Campanhas Personalizadas
Personalize campanhas incluindo nomes de destinatários, valores de pagamento ou qualquer outra informação relevante no arquivo Excel.

### Exibição Top 5 Linhas
Após o upload do arquivo Excel, o aplicativo exibe as 5 primeiras linhas para conveniência do usuário.

### Selecção da Coluna
Prossiga para selecionar a coluna contendo números móveis e digite o ID do remetente e o conteúdo da mensagem.

### Inserir Posições
Utilizar o **"Inserir substituto"** opção de substituir placeholders por valores reais do arquivo Excel no conteúdo da mensagem.

![Insert Placeholders](images/smsexcel3.png)

### Antevisão
Clique na **"Preview"** botão para ver o topo 5 mensagens SMS personalizadas, juntamente com contagens de caracteres.

![Preview](images/smsexcel4.png)

### Agendamento
Os usuários podem agendar campanhas para uma data específica. O sistema é padrão para o fuso horário configurado no perfil do usuário.

### Suporte ao Unicode
Para o conteúdo da mensagem Unicode, os usuários podem ativar a opção Unicode.

### Antevisão Final e Envio
Após clicar no **"Enviar"** botão, o aplicativo fornece uma tela de visualização com detalhes como país, custo total, etc. Confirmando a tela de visualização de mensagens envia as campanhas.

![Final Preview](images/smsexcel5.png)

---

## Orientações de Utilização

- **Conteúdo do Excel**: Organize números móveis e detalhes adicionais no arquivo Excel.
- **Selecção da Coluna**: Escolha a coluna contendo números móveis e digite ID do remetente e conteúdo da mensagem.
- **Inserir Posições**: Inserir placeholders dinamicamente para conteúdo personalizado.
- **Antevisão e Agendamento**: Veja as 5 principais mensagens, verifique a contagem de caracteres e agendar campanhas se necessário.
- **Opção Unicode**: Active Unicode para conteúdo de mensagem de script não-latino.
- **Confirmação Final**: Reveja detalhes na tela de visualização antes de confirmar e enviar as campanhas.

![Usage Guidelines](images/smsexcel6.png)

A **SMS do Excel** recurso em *iTextPro* simplifica o processo de lançamento de campanhas com conteúdo personalizado, tornando-o eficiente e fácil de usar. Seja agendando campanhas ou utilizando suporte Unicode, esta ferramenta oferece flexibilidade para diversas necessidades de comunicação.
