
# SMS de Excel: Campagnes de déclenchement avec des fichiers Excel

Les **SMS depuis Excel** onglet *iTextPro* offre aux utilisateurs une approche simplifiée pour déclencher des campagnes à l'aide d'un fichier Excel contenant divers messages ou variables. Cette fonctionnalité est particulièrement utile pour envoyer des rappels, des offres promotionnelles personnalisées, et plus encore.

![SMS From Excel](images/smsexcel1.png)

## Caractéristiques principales

### Téléchargement de fichier Excel
Les utilisateurs peuvent déclencher des campagnes à partir d'un fichier Excel contenant des numéros mobiles et des détails supplémentaires.

![Excel File Upload](images/smsexcel2.png)

### Télécharger un exemple de fichier
Cliquez sur le **"Voir l'échantillon"** onglet pour télécharger un exemple de fichier Excel, fournissant des conseils sur le format requis.

### Campagnes personnalisées
Personnaliser les campagnes en incluant les noms de bénéficiaires, les montants de paiement ou toute autre information pertinente dans le fichier Excel.

### Affichage des 5 premières lignes
Après avoir téléchargé le fichier Excel, l'application affiche les 5 premières lignes pour faciliter l'utilisation.

### Sélection des colonnes
Sélectionnez la colonne contenant les numéros mobiles et entrez l'identifiant de l'expéditeur et le contenu du message.

### Insérer les titulaires
Utiliser **"Insérer le détenteur du lieu"** option de remplacer les détenteurs de place par des valeurs réelles à partir du fichier Excel dans le contenu du message.

![Insert Placeholders](images/smsexcel3.png)

### Aperçu
Cliquez sur **Aperçu** bouton pour afficher les 5 messages SMS personnalisés en haut avec le nombre de caractères.

![Preview](images/smsexcel4.png)

### Calendrier
Les utilisateurs peuvent programmer des campagnes pour une date précise. Le système est par défaut sur le fuseau horaire configuré dans le profil de l'utilisateur.

### Prise en charge Unicode
Pour le contenu du message Unicode, les utilisateurs peuvent activer l'option Unicode.

### Aperçu et envoi finals
Après avoir cliqué sur **"Envoyer"** bouton, l'application fournit un écran d'aperçu avec des détails tels que le pays, le coût total, etc. Confirmer l'écran d'aperçu du message envoie les campagnes.

![Final Preview](images/smsexcel5.png)

---

## Lignes directrices pour l'utilisation

- **Contenu Excel**: Organisez des numéros mobiles et des détails supplémentaires dans le fichier Excel.
- **Sélection des colonnes**: Choisissez la colonne contenant les numéros mobiles et entrez l'identifiant de l'expéditeur et le contenu du message.
- **Insérer les titulaires**: Insérez dynamiquement des supports pour un contenu personnalisé.
- **Aperçu et calendrier**: Visualisez les 5 premiers messages, vérifiez le nombre de caractères et planifiez des campagnes si nécessaire.
- **Option Unicode**: Activer Unicode pour les contenus de script non latins.
- **Confirmation finale**: Examiner les détails dans l'écran de prévisualisation avant de confirmer et d'envoyer les campagnes.

![Usage Guidelines](images/smsexcel6.png)
