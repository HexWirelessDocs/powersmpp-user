
# Gérer le modèle : Configurer le contenu du message statique

Les **Gérer le modèle** fonctionnalité dans iTextPro permet aux utilisateurs de configurer efficacement le contenu du message statique pour leurs campagnes SMS. Cette section fournit un guide complet sur la façon dont les utilisateurs peuvent gérer les modèles au sein de la plateforme.

![Manage Template](images/template1.png)

## Fonctionnalité principale

### Afficher et approuver les modèles
- Les utilisateurs peuvent accéder à une liste affichant tous les modèles ainsi que leurs statuts d'approbation respectifs. 
- **Remarque:** Si le compte est activé pour **Ouvrir le modèle**, les utilisateurs peuvent choisir de sauter cette configuration.

---

### Création d'un nouveau modèle
Les utilisateurs peuvent lancer la création d'un nouveau modèle en cliquant sur le **"Ajouter un modèle"** onglet. Cette action ouvre une boîte popup pour la configuration.

#### Étapes pour ajouter un modèle :
1. **Cliquez sur "Ajouter un modèle"** 
 Accédez à la section « Gérer le modèle » et cliquez sur l'onglet « Ajouter un modèle ».

2. **Boîte de configuration popup** 
 Une boîte de configuration apparaîtra, permettant aux utilisateurs de configurer le nouveau modèle.

3. **Détails de configuration** 
 Les utilisateurs peuvent saisir les détails nécessaires pour le nouveau modèle, conformément aux lignes directrices fournies dans la boîte de configuration.

4. **Insérer des titulaires de place** 
 Les utilisateurs peuvent insérer des placeholders en cliquant sur le **"Insert Placeholder"** bouton. 
   > Tout autre que le détenteur de place sera traité comme un contenu statique et ne pourra être édité lors de l'envoi de messages.

5. **Processus d'approbation** 
 Enregistrer les détails du modèle configuré. Selon le processus d'approbation configuré par l'administrateur, le modèle peut passer par une étape d'approbation.

---

### Liste des modèles
- Les utilisateurs peuvent voir la liste complète des modèles, y compris leur statut d'approbation, dans la **Gérer le modèle** Chapitre.

---

### Note sur le modèle ouvert
Si le compte est activé pour **Ouvrir le modèle**, les utilisateurs ont la flexibilité de sauter le processus de configuration de gabarit.

---

## Rappel important
Une fois qu'un modèle est approuvé, les utilisateurs ne peuvent modifier la portion de placeholder que lorsqu'ils envoient des messages. 
Les utilisateurs utilisant les API doivent faire preuve de prudence, car toute divergence dans l'espacement, les virgules ou les arrêts complets manquants peut entraîner un modèle invalide.

---

Les **Gérer le modèle** fonctionnalité dans iTextPro assure une configuration efficace du contenu du message statique, offrant aux utilisateurs une flexibilité et un contrôle sur leurs campagnes SMS. Que ce soit l'approbation de modèles existants ou la création de nouveaux modèles, cet outil contribue à une expérience de messagerie transparente et personnalisée.
