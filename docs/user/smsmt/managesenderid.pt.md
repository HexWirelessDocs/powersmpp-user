
# Gerenciar o ID do remetente: Configurar endereços de origem

iTextPro's **Gerenciar o ID do remetente** o recurso capacita os usuários a configurar e supervisionar de forma eficiente os endereços de origem para suas campanhas SMS. Esta seção fornece uma visão geral de como os usuários podem gerenciar IDs de remetentes dentro da plataforma.

![Manage Sender ID](images/senderid1.png)

## Funcionalidade chave

### Ver e Aprovar IDs do remetente
- Os usuários podem acessar uma lista exibindo todos os IDs do remetente juntamente com seus respectivos status de aprovação. 
- **Nota:** Se a conta estiver habilitada para *Abrir o ID do remetente*, os usuários podem optar por pular esta configuração.

### Criando um novo ID do remetente
Os usuários podem iniciar a criação de um novo ID de remetente clicando no **"Adicionar ID do remetente"** tab. Esta ação abre uma caixa de contexto para configuração.

#### Passos para Adicionar um ID do remetente

1. **Clique em "Adicionar ID do remetente" Tab** 
 Acesso *Gerenciar o ID do remetente* section e clique na **"Adicionar ID do remetente"** tab.

2. **Caixa de Configuração do Popup** 
 Uma caixa de configuração aparecerá, permitindo aos usuários configurar o novo ID do remetente.

3. **Detalhes da Configuração** 
 Os usuários podem inserir os detalhes necessários para o novo ID do remetente, seguindo as diretrizes fornecidas na caixa de configuração.

4. **Processo de gravação e aprovação** 
 Gravar os detalhes do ID do remetente configurados. Dependendo do processo de aprovação configurado pelo administrador, o ID do remetente pode passar por uma fase de aprovação.

### Lista de IDs do remetente
Os usuários podem visualizar a lista completa de IDs de remetentes, incluindo seus status de aprovação, no **Gerenciar o ID do remetente** secção.

### Nota ao Abrir o ID do remetente
Se a conta estiver habilitada para *Abrir o ID do remetente*, os usuários têm a flexibilidade para pular o processo de configuração de ID do remetente.

---

A **Gerenciar o ID do remetente** recurso no iTextPro simplifica o processo de configuração e supervisão de endereços de origem para campanhas SMS. Os usuários podem facilmente adicionar novos IDs de remetentes, rever seus status de aprovação e garantir o cumprimento dos requisitos da plataforma.
