
# Compor SMS: Campanhas SMS A2P

In **iTextPro**, **Compor SMS** característica dentro da **SMS MT** opção fornece aos usuários com uma ferramenta poderosa para criar e enviar campanhas SMS A2P (Aplicação à Pessoa). Aqui está um guia detalhado sobre como usar este recurso:

![Compose SMS Page](images/compose1.png)

## Abrir a Página de SMS
Clique na **SMS MT** opção para abrir a página Compor SMS.

## Detalhes da Campanha

### a. Nome da campanha
Digite um nome amigável para sua campanha. Por padrão, um nome é gerado automaticamente com a hora atual e o prefixo <span data-ph="0"></span>.

### b. ID do remetente
Selecione um ID de remetente aprovado no menu suspenso. 
- Se o administrador tiver desactivado o **Abrir a Opção de ID do remetente**, os utilizadores devem escolher entre a lista fornecida. 
- Se habilitado, os usuários podem inserir um ID de remetente dinâmico em uma caixa de texto. Este ID aparecerá como endereço do remetente no celular do destinatário.

### c. Contactos
Escolha contatos de:
- Grupos armazenados
- Enviar ficheiros locais
- Indique manualmente os contactos
- Copia- os para a caixa de texto

Assegurar que os números móveis são separados por vírgulas, começando pelo código do país (sem <span data-ph="0"></span>). Opcionalmente, marque a caixa para adicionar automaticamente o código do país.

### d. Digite a Mensagem
Use a caixa de texto para inserir a mensagem. 
- O sistema exibe as últimas 5 mensagens para referência rápida. 
- Um contador na parte inferior indica o comprimento da mensagem e a contagem.

### e. Rascunho e modelos
Selecione conteúdo de rascunhos ou modelos existentes. 
- **Rascunhos** pode ser editado. 
- **Modelos** Permitir modificações apenas aos placeholders.

### f. Flash
Desmarcar ou verificar o **Flash** caixa para controlar se a mensagem aparece diretamente na tela do aparelho.

### g. Unicode
O aplicativo detecta mensagens Unicode automaticamente e verifica a caixa Unicode.

### h. Calendário
Salvar como um rascunho ou agendar uma campanha futura. 
O sistema usa o fuso horário configurado no perfil do usuário por padrão.

### i. Enviar SMS
Clicando em **Enviar SMS** fornece uma visualização dos custos e outros detalhes antes de finalizar a campanha.

---

## Nota sobre Contadores
Contadores de mensagens e caracteres são ferramentas indicativas devido à compatibilidade do navegador e problemas de codificação. 
Recomenda-se testar o conteúdo em alguns números antes de campanhas em larga escala.

---

## Rascunhos e Modelos
Use rascunhos salvos ou modelos para eficiência. 
- Os rascunhos podem ser editados. 
- Os modelos permitem a modificação de espaços.

## Flash e Unicode
- Utilize a opção Flash para visibilidade imediata na tela do aparelho. 
- Habilitar Unicode para mensagens em scripts não- latinos.

## Agendamento
Salvar rascunhos ou agendar campanhas futuras. O sistema usa o fuso horário do usuário por padrão.

## Enviar SMS
Clicando **Enviar SMS** fornece uma visualização abrangente dos custos e detalhes antes do lançamento final da campanha.

---

A **Compor SMS** O recurso no iTextPro capacita os usuários a criar campanhas SMS personalizadas e eficientes A2P. 
Desde a personalização de IDs de remetentes até campanhas de agendamento, esta ferramenta proporciona flexibilidade e controle sobre todos os aspectos da comunicação SMS.
