
# Composez SMS : Campagnes SMS A2P

Dans **iTextPro**, les **Composer un SMS** caractéristique de la **Autres** option fournit aux utilisateurs un outil puissant pour créer et envoyer des campagnes SMS A2P (Application-to-Person). Voici un guide détaillé sur la façon d'utiliser cette fonctionnalité :

![Compose SMS Page](images/compose1.png)

## Ouvrir la page de SMS Compose
Cliquez sur le **Autres** possibilité d'ouvrir la page Compose SMS.

## Détails de la campagne

### a. Nom de la campagne
Entrez un nom amical pour votre campagne. Par défaut, un nom est automatiquement généré avec la date actuelle et le préfixe <span data-ph="0"></span>.

### b. Identification de l'expéditeur
Sélectionnez un identifiant d'expéditeur approuvé dans le menu déroulant. 
- Si l'administrateur a désactivé **Ouvrir l'option ID de l'expéditeur**, les utilisateurs doivent choisir dans la liste fournie. 
- Si activé, les utilisateurs peuvent saisir un identifiant d'expéditeur dynamique dans une zone de texte. Cet ID apparaîtra comme adresse de l'expéditeur dans le mobile du destinataire.

### c. Contacts
Choisissez des contacts parmi :
- Groupes stockés
- Télécharger les fichiers locaux
- Saisissez manuellement les contacts
- Copiez-les dans la boîte de texte

Veiller à ce que les numéros mobiles soient séparés par des virgules, en commençant par le code pays (sans <span data-ph="0"></span>). En option, cochez la case pour ajouter automatiquement le code pays.

### d. Entrez le message
Utilisez la zone de texte pour saisir le message. 
- Le système affiche les 5 derniers messages pour une référence rapide. 
- Un compteur en bas indique la longueur et le nombre du message.

### e. Projet et modèles
Sélectionnez le contenu des ébauches ou des modèles existants. 
- **Projets** peut être édité. 
- **Modèles** permettre des modifications uniquement aux détenteurs de place.

### f. Flash
Décochez ou vérifiez la **Flash** box pour contrôler si le message apparaît directement sur l'écran du combiné.

### g. Unicode
L'application détecte automatiquement les messages Unicode et cochez la case Unicode.

### h. Calendrier
Enregistrer en tant que projet ou programmer une campagne future. 
Le système utilise par défaut le fuseau horaire configuré dans le profil de l'utilisateur.

### i. Envoyer des SMS
Cliquez sur **Envoyer un SMS** fournit un aperçu des coûts et autres détails avant de finaliser la campagne.

---

## Note sur les compteurs
Les compteurs de messages et de caractères sont des outils indicatifs en raison de la compatibilité du navigateur et des problèmes d'encodage. 
Il est recommandé de tester le contenu sur quelques numéros avant les campagnes à grande échelle.

---

## Projets et modèles
Utilisez des ébauches ou des modèles sauvegardés pour l'efficacité. 
- Les projets peuvent être édités. 
- Les modèles permettent de modifier les détenteurs de place.

## Flash et Unicode
- Utilisez l'option Flash pour une visibilité immédiate sur l'écran du combiné. 
- Activer Unicode pour les messages dans les scripts non latins.

## Calendrier
Enregistrer les ébauches ou programmer les campagnes futures. Le système utilise le fuseau horaire de l'utilisateur par défaut.

## Envoyer un SMS
Cliquer **Envoyer un SMS** fournit un aperçu complet des coûts et des détails avant le lancement final de la campagne.

---

Les **Composer un SMS** fonctionnalité dans iTextPro permet aux utilisateurs de créer des campagnes SMS A2P personnalisées et efficaces. 
De la personnalisation des ID de l'expéditeur aux campagnes de planification, cet outil offre flexibilité et contrôle sur tous les aspects de la communication SMS.
