
# Gérer l'ID de l'expéditeur : Configurer les adresses sources

iTextPro **Gérer l'identifiant de l'expéditeur** fonctionnalité permet aux utilisateurs de configurer et de superviser efficacement les adresses sources pour leurs campagnes SMS. Cette section donne un aperçu de la façon dont les utilisateurs peuvent gérer les ID de l'expéditeur au sein de la plateforme.

![Manage Sender ID](images/senderid1.png)

## Fonctionnalité principale

### Afficher et approuver les identifiants de l'expéditeur
- Les utilisateurs peuvent accéder à une liste affichant tous les ID de l'expéditeur ainsi que leurs statuts d'approbation respectifs. 
- **Remarque:** Si le compte est activé pour *Ouvrir l'identifiant de l'expéditeur*, les utilisateurs peuvent choisir de sauter cette configuration.

### Création d'un nouvel identifiant d'expéditeur
Les utilisateurs peuvent lancer la création d'un nouvel ID d'expéditeur en cliquant sur le **"Ajouter l'identifiant de l'expéditeur"** onglet. Cette action ouvre une boîte popup pour la configuration.

#### Étapes à suivre pour ajouter un identifiant d'expéditeur

1. **Cliquez sur "Ajouter l'identifiant de l'expéditeur"** 
 Accès *Gérer l'identifiant de l'expéditeur* section et cliquez sur **"Ajouter l'identifiant de l'expéditeur"** onglet.

2. **Boîte de configuration popup** 
 Une boîte de configuration apparaîtra, permettant aux utilisateurs de configurer le nouvel ID de l'expéditeur.

3. **Détails de configuration** 
 Les utilisateurs peuvent saisir les détails nécessaires pour le nouvel identifiant de l'expéditeur, conformément aux directives fournies dans la boîte de configuration.

4. **Processus d'enregistrement et d'approbation** 
 Enregistrer les détails de l'identifiant de l'expéditeur configurés. Selon le processus d'approbation configuré par l'administrateur, l'identifiant de l'expéditeur peut passer par une étape d'approbation.

### Liste des ID de l'expéditeur
Les utilisateurs peuvent voir la liste complète des ID de l'expéditeur, y compris leur statut d'approbation, dans le **Gérer l'identifiant de l'expéditeur** Chapitre.

### Note sur l'identifiant de l'expéditeur ouvert
Si le compte est activé pour *Ouvrir l'identifiant de l'expéditeur*, les utilisateurs ont la flexibilité de sauter le processus de configuration de l'identifiant de l'expéditeur.

---

Les **Gérer l'identifiant de l'expéditeur** fonctionnalité dans iTextPro rationalise le processus de configuration et de supervision des adresses sources pour les campagnes SMS. Les utilisateurs peuvent facilement ajouter de nouveaux identifiants d'expéditeur, examiner leur statut d'approbation et s'assurer de la conformité aux exigences de la plateforme.
