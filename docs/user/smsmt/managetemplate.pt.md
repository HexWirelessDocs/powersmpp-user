
# Gerir o Modelo: Configurar o Conteúdo da Mensagem Estática

A **Gerenciar Modelo** recurso no iTextPro capacita os usuários a configurar eficientemente conteúdo de mensagem estática para suas campanhas SMS. Esta seção fornece um guia abrangente sobre como os usuários podem gerenciar modelos dentro da plataforma.

![Manage Template](images/template1.png)

## Funcionalidade chave

### Ver e Aprovar Modelos
- Os usuários podem acessar uma lista exibindo todos os modelos, juntamente com seus respectivos status de aprovação. 
- **Nota:** Se a conta estiver habilitada para **Abrir Modelo**, os usuários podem optar por pular esta configuração.

---

### Criar um Novo Modelo
Os usuários podem iniciar a criação de um novo modelo clicando no **"Adicionar Modelo"** tab. Esta ação abre uma caixa de contexto para configuração.

#### Passos para Adicionar um Modelo:
1. **Clique na aba "Adicionar modelo"** 
 Acesse a seção "Gerenciar Modelo" e clique na guia "Adicionar Modelo".

2. **Caixa de Configuração do Popup** 
 Aparecerá uma caixa de configuração, permitindo aos usuários configurar o novo modelo.

3. **Detalhes da Configuração** 
 Os usuários podem inserir os detalhes necessários para o novo modelo, seguindo as diretrizes fornecidas na caixa de configuração.

4. **Inserir os Positivos** 
 Os usuários podem inserir placeholders clicando no **"Inserir Titular"** Botão. 
   > Qualquer outra coisa que não seja o placeholder será tratada como conteúdo estático e não poderá ser editado durante o envio de mensagens.

5. **Processo de aprovação** 
 Gravar os detalhes do modelo configurados. Dependendo do processo de aprovação configurado pelo administrador, o modelo pode passar por uma fase de aprovação.

---

### Lista de Modelos
- Os utilizadores podem visualizar a lista completa de modelos, incluindo os seus estados de aprovação, no **Gerenciar Modelo** secção.

---

### Nota sobre o Modelo Aberto
Se a conta estiver habilitada para **Abrir Modelo**, os usuários têm a flexibilidade para pular o processo de configuração do modelo.

---

## Lembrete Importante
Uma vez aprovado um modelo, os usuários só podem editar a porção de placeholder enquanto enviam mensagens. 
Usuários que utilizam APIs precisam ter cuidado, pois quaisquer discrepâncias no espaçamento, vírgulas ou falta de paradas completas podem resultar em um modelo inválido.

---

A **Gerenciar Modelo** O recurso no iTextPro garante uma configuração eficiente do conteúdo da mensagem estática, proporcionando aos usuários flexibilidade e controle sobre suas campanhas SMS. Quer aprove modelos existentes ou crie novos, esta ferramenta contribui para uma experiência de mensagens perfeita e personalizada.
