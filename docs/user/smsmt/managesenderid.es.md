
# Administrar ID del remitente: Configurar direcciones de origen

iTextPro's **Administrar ID del remitente** La función permite a los usuarios configurar y supervisar eficientemente las direcciones de origen para sus campañas SMS. Esta sección ofrece una visión general de cómo los usuarios pueden gestionar ID de remitente dentro de la plataforma.

![Manage Sender ID](images/senderid1.png)

## Función clave

### View and Approve Sender IDs
- Los usuarios pueden acceder a una lista que muestre todos los IDs del remitente junto con sus respectivos estados de aprobación. 
- **Nota:** Si la cuenta está habilitada *Open Sender ID*, los usuarios pueden optar por saltar esta configuración.

### Crear un nuevo ID de remitente
Los usuarios pueden iniciar la creación de un nuevo ID del remitente haciendo clic en el **"Agregar ID de remitente"** tab. Esta acción abre una caja emergente para la configuración.

#### Pasos para agregar un ID de remitente

1. **Haga clic en "Añadir ID de remitente" Tab** 
 Acceso *Administrar ID del remitente* sección y haga clic en **"Agregar ID de remitente"** tab.

2. **Popup Configuration Box** 
 Aparecerá un cuadro de configuración, permitiendo a los usuarios configurar el nuevo ID del remitente.

3. **Detalles de configuración** 
 Los usuarios pueden introducir los detalles necesarios para el nuevo ID del remitente, siguiendo las directrices proporcionadas en el cuadro de configuración.

4. **Proceso de ahorro y aprobación** 
 Guarde los datos de identificación del remitente configurados. Dependiendo del proceso de aprobación configurado por el administrador, el ID del remitente puede pasar por una etapa de aprobación.

### List of Sender IDs
Los usuarios pueden ver la lista completa de ID de remitente, incluyendo sus estados de aprobación, en la **Administrar ID del remitente** sección.

### Nota sobre Open Sender ID
Si la cuenta está habilitada *Open Sender ID*, los usuarios tienen la flexibilidad para saltar el proceso de configuración de ID del remitente.

---

El **Administrar ID del remitente** característica en iTextPro simplifica el proceso de configuración y supervisión de las direcciones de origen para las campañas SMS. Los usuarios pueden agregar fácilmente nuevos ID de remitente, revisar sus estados de aprobación y garantizar el cumplimiento de los requisitos de la plataforma.
