
# SMS From Excel: Campañas de Triggering con archivos Excel

El **SMS From Excel** pestaña en *iTextPro* proporciona a los usuarios un enfoque simplificado para desencadenar campañas utilizando un archivo Excel que contiene varios contenidos de mensajes o variables. Esta característica es particularmente útil para enviar recordatorios, ofertas promocionales personalizadas y más.

![SMS From Excel](images/smsexcel1.png)

## Características clave

### Excel File Upload
Los usuarios pueden activar campañas desde un archivo Excel que contiene números móviles y detalles adicionales.

![Excel File Upload](images/smsexcel2.png)

### Descarga de archivos de muestra
Haga clic en **"Ver Muestra"** pestaña para descargar un archivo de muestra Excel, proporcionando orientación sobre el formato requerido.

### Campañas personalizadas
Personalizar campañas incluyendo nombres de destinatarios, cantidades de pago o cualquier otra información relevante en el archivo Excel.

### Top 5 Rows Display
Después de subir el archivo Excel, la aplicación muestra las 5 filas superiores para la comodidad del usuario.

### Selección de columnas
Proceda para seleccionar la columna que contiene números móviles e introducir el ID del remitente y el contenido del mensaje.

### Insertar marcadores de posición
Utilizar el **"Insert placeholder"** opción para reemplazar los marcadores de posición con valores reales del archivo Excel en el contenido del mensaje.

![Insert Placeholders](images/smsexcel3.png)

### Vista previa
Haga clic en **"Preview"** botón para ver la parte superior 5 mensajes SMS personalizados junto con los números de caracteres.

![Preview](images/smsexcel4.png)

### Planificación
Los usuarios pueden programar campañas para una fecha específica. El sistema predetermina la zona horaria configurada en el perfil del usuario.

### Unicode Support
Para contenidos de mensaje Unicode, los usuarios pueden habilitar la opción Unicode.

### Vista previa final y enviar
Después de hacer clic en **"Enviar"** botón, la aplicación proporciona una pantalla de vista previa con detalles como país, costo total, etc. Confirmando la pantalla de vista previa del mensaje envía las campañas.

![Final Preview](images/smsexcel5.png)

---

## Directrices de uso

- **Excel Content**: Organizar números móviles y detalles adicionales en el archivo Excel.
- **Selección de columnas**: Elija la columna que contiene números móviles e introduzca el ID del remitente y el contenido del mensaje.
- **Insertar marcadores de posición**: Inserción dinámica para el contenido personalizado.
- **Vista previa y programa**: Vea los 5 mensajes principales, compruebe la cuenta de caracteres y programe campañas si es necesario.
- **Opción Unicode**: Habilitar Unicode para contenidos de mensajes de script no latinos.
- **Confirmación final**: Revisar detalles en la pantalla de vista previa antes de confirmar y enviar las campañas.

![Usage Guidelines](images/smsexcel6.png)

El **SMS From Excel** característica en *iTextPro* simplifica el proceso de lanzamiento de campañas con contenido personalizado, haciéndolo eficiente y fácil de usar. Ya sea programando campañas o utilizando soporte Unicode, esta herramienta ofrece flexibilidad para diversas necesidades de comunicación.
