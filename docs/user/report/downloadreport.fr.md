
## Télécharger le rapport : Accès et demandes de rapports

Les **Télécharger le rapport** onglet dans iTextPro agit comme un centre de gestion des rapports. 
Il permet aux utilisateurs de **accès aux rapports demandés antérieurement** et **lancer de nouvelles demandes de rapport**.

![Download Report](images/download1.png)

---

### Caractéristiques principales

- **Visibilité des rapports demandés** 
 Tous les rapports demandés précédemment sont énumérés sous la rubrique **Télécharger le rapport** onglet pour un accès rapide.

- **Demander de nouveaux rapports** 
 Utilisez la **Rapport de demande** l'option de générer de nouveaux rapports au besoin.

- **Indication du statut en attente** 
 Rapports marqués comme *En attente* signifie que le système récupère toujours les données requises. 
 Ceux-ci seront disponibles pour téléchargement une fois le traitement terminé.

---

### Étapes d'accès et rapports de demande

1. **Rapport de téléchargement d'accès** 
 Naviguez vers la **Télécharger le rapport** onglet dans iTextPro.

2. **Consulter les rapports demandés** 
 Consultez la liste des rapports déjà demandés. 
 Cliquez sur le rapport souhaité pour le télécharger.

3. **Demander un nouveau rapport** 
 Utilisez la **Rapport de demande** bouton pour lancer une nouvelle demande de rapport.

4. **Surveillance en cours** 
 Si le rapport est marqué comme suit: *En attente*, attendez qu'il soit traité et devient disponible pour le téléchargement.
