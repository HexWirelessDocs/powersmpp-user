
## Relatório de Download: Acesso e Solicitação de Relatórios

A **Baixar relatório** tab no iTextPro atua como um hub central para gerenciar relatórios. 
Permite aos utilizadores **acesso aos relatórios anteriormente solicitados** e **iniciar novos pedidos de relatório**.

![Download Report](images/download1.png)

---

### Características Principais

- **Visibilidade dos relatórios solicitados** 
 Todos os relatórios anteriormente solicitados estão listados na **Baixar relatório** aba para acesso rápido.

- **Solicitar Novos Relatórios** 
 Utilizar o **Relatório de Pedido** opção para gerar novos relatórios conforme necessário.

- **Indicação do estado pendente** 
 Relatórios assinalados como *Pendente* significa que o sistema continua a obter os dados necessários. 
 Estes estarão disponíveis para download uma vez que o processamento estiver concluído.

---

### Passos para o acesso e solicitação de relatórios

1. **Relatório de Transferência de Acesso** 
 Navegar para o **Baixar relatório** tab no iTextPro.

2. **Ver Relatórios Solicitados** 
 Navegue na lista de relatórios solicitados anteriormente. 
 Clique no relatório desejado para baixá-lo.

3. **Solicitar um Novo Relatório** 
 Utilizar o **Relatório de Pedido** botão para iniciar um novo pedido de relatório.

4. **Monitorar estado pendente** 
 Se o relatório estiver marcado como *Pendente*, aguarde até que seja processado e fique disponível para download.
