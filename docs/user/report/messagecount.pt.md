
## Contagem de mensagens: Análise de Tráfego SMS com o Estado

A **Contagem de Mensagens** recurso no iTextPro fornece uma análise detalhada, em termos de status do tráfego SMS. 
Permite aos usuários monitorar o desempenho de entrega e as tendências de uso durante um período de tempo selecionado.

![Message Count](images/messagecount1.png)

---

### Características Principais

- **Contagem de Estado** 
 Ver as contagens de mensagens categorizadas por status (por exemplo, Entregue, Pendente, Falha). 
 Ajuda a identificar o desempenho da entrega e distribuição de tráfego.

- **Selecção do Intervalo de Datas** 
 Selecionar um específico **intervalo de datas** para ver a mensagem conta para esse período.

- **Resumo mensal com Drill-down** 
 Quando um mês específico é selecionado, o sistema exibe **total SMS/Usage count** por omissão. 
 A **hiperlink** permite perfurar para baixo em um detalhado **relatório de resumo date-wise**.

- **Precisão do fuso horário** 
 Todas as contagens de mensagens são exibidas de acordo com o fuso horário configurado pelo usuário.

---

### Passos para Ver a Contagem de Mensagens

1. **Contagem de Mensagens de Acesso** 
 Navegar para o **Contagem de Mensagens** secção em iTextPro.

2. **Selecionar intervalo de datas** 
 Escolha o intervalo de data ou mês desejado.

3. **Explorar o Resumo Mensal** 
 Se estiver seleccionado um mês, clique no hiperlink para ver um detalhe **desagregação dia-a-dia**.

---

**Dica:** Use o intervalo de datas e filtros de status juntos para análise de tráfego mais direcionada.
