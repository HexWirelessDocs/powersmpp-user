
## Nombre de messages : Analyse du trafic SMS par statut

Les **Nombre de messages** fonctionnalité dans iTextPro fournit une analyse détaillée, selon le statut du trafic SMS. 
Il permet aux utilisateurs de suivre les performances et les tendances d'utilisation sur une période donnée.

![Message Count](images/messagecount1.png)

---

### Caractéristiques principales

- **Dénombrement par statut** 
 Afficher le nombre de messages classés par état (p. ex., livré, en attente, échoué). 
 Aide à identifier les performances de livraison et la distribution du trafic.

- **Sélection de l'intervalle de date** 
 Sélectionner une donnée **date** voir le message compte pour cette période.

- **Sommaire mensuel avec la réduction des forages** 
 Lorsqu'un mois spécifique est sélectionné, le système affiche **Nombre total de SMS/Usage** par défaut. 
 A **Lien hypertexte** permet le forage vers le bas dans un détail **Rapport de synthèse sur la date**.

- **Précision des fuseaux horaires** 
 Tous les nombres de messages sont affichés en fonction du fuseau horaire configuré par l'utilisateur.

---

### Étapes pour voir le nombre de messages

1. **Nombre de messages d'accès** 
 Naviguez vers la **Nombre de messages** dans iTextPro.

2. **Sélectionner la plage de dates** 
 Choisissez la date ou le mois souhaité.

3. **Explorer le sommaire mensuel** 
 Si un mois est sélectionné, cliquez sur l'hyperlien pour afficher un **Répartition au jour le jour**.

---

**Conseil :** Utilisez la plage de dates et les filtres d'état ensemble pour une analyse du trafic plus ciblée.
