
## Mi agenda

El **Mi agenda** característica en iTextPro simplifica el seguimiento y la gestión de las campañas programadas. 
Permite a los usuarios:
- Supervisar el estado de las campañas programadas
- Realizar ajustes en las campañas pendientes
- Garantizar la ejecución oportuna y precisa de las campañas

![My Schedule](images/schedule1.png)

---

### Programación de una campaña pendiente
Si un usuario desea **reprogramar una campaña pendiente y programada**:
1. Navigate a la **Medida** ficha para la campaña deseada.
2. Haga clic en **Editar** botón.
3. Seleccione el apropiado **Zona horaria** y **nuevo horario**.
4. Guardar cambios para confirmar el calendario actualizado.

![Reschedule Campaign](images/schedule2.png)

---

**Sugerencia:** Verifique siempre la zona horaria seleccionada para asegurar que la campaña envíe a la hora local prevista para los receptores.
