
# Relatórios: Análise abrangente das mensagens MT

A **Relatórios** tab no iTextPro fornece aos usuários um conjunto abrangente de ferramentas para analisar e rastrear **MT (Terminado móvel)** mensagens. 
Inclui características como **Relatórios de Campanha**, **Agendar Campanhas**, **Contagem de Mensagens**, e **Obter relatórios**.

---

## Relatórios de Campanha

### Área de Campanha
Mostra uma lista consolidada de campanhas executadas pelo usuário. 
Os usuários podem acessar rapidamente os principais detalhes da campanha, incluindo:
- Estado da campanha
- Taxas de entrega
- Datas de execução
- Outras estatísticas relevantes

![Campaign View](images/campaign1.png)

---

### Vista da Lista
Apresenta um **DLR (Receita de Entrega)** apresentar relatório para campanhas executadas. 
Os usuários podem rever informações detalhadas do estado de entrega para cada mensagem dentro de uma campanha.

---

### Estado da Campanha
- Navegar para **Mostrar o Estado da Campanha** para ver a fila de mensagens e acompanhar o progresso da campanha. 
- Fornece uma visão em tempo real das entregas concluídas e pendentes.

---

### Parar uma Campanha
Se necessário, as campanhas podem ser interrompidas no meio do processo:
1. Vá para a campanha desejada.
2. Clique na **Parar** Botão.

---

### Página de Acções
Oferece conta de status de mensagem para campanhas executadas, ajudando os usuários a analisar:
- Mensagens enviadas
- Mensagens pendentes
- Mensagens Falhadas
- Outras categorias de estatuto

![Actions Tab](images/campaign2.png)

---

### Baixar relatório
Permite baixar o relatório completo da campanha em **Excel (.xls)** formato para:
- Revisão offline
- Objectivos do arquivo
- Análise avançada dos dados

---

### Reenviar Página
Permite reenviar mensagens de campanha com base no seu estado (por exemplo, falhou ou está pendente).

**Reenviar Elegibilidade:**

- • Disponível para **campanhas executadas** 
- Não disponível para: **campanhas em fila de espera** ou **mensagens longas ainda em andamento**

---

**Dica:** Utilizar o **Baixar relatório** opção antes de reenviar para validar a lista de alvos e evitar entregas duplicadas.
