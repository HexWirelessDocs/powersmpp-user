
## Informe de descarga: Acceso y Solicitud de informes

El **Descargar Report** tab in iTextPro actúa como un centro central para gestionar los informes. 
Permite a los usuarios **antes solicitados** y **iniciar nuevas solicitudes de informes**.

![Download Report](images/download1.png)

---

### Características clave

- **Visibilidad de los informes solicitados** 
 Todos los informes solicitados anteriormente figuran en la lista **Descargar Report** ficha para acceso rápido.

- **Solicitar nuevos informes** 
 Usar el **Solicitud de informe** opción para generar nuevos informes según sea necesario.

- **Indicación del estado pendiente** 
 Informes marcados como *Pendiente* significa que el sistema sigue buscando los datos necesarios. 
 Estos estarán disponibles para su descarga una vez que el procesamiento esté completo.

---

### Pasos para acceder y solicitar informes

1. **Access Download Report** 
 Navigate a la **Descargar Report** ficha en iTextPro.

2. **View Requested Reports** 
 Examine la lista de informes previamente solicitados. 
 Haga clic en el informe deseado para descargarlo.

3. **Solicitar un nuevo informe** 
 Usar el **Solicitud de informe** botón para iniciar una nueva solicitud de informe.

4. **Monitor Pending Status** 
 Si el informe está marcado como *Pendiente*, espere hasta que se procesa y se pone disponible para su descarga.
