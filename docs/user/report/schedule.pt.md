
## Minha agenda

A **Minha agenda** o recurso no iTextPro simplifica o rastreamento e gerenciamento de campanhas agendadas. 
Permite aos utilizadores:
- Monitorar o status das campanhas agendadas
- Fazer ajustes nas campanhas pendentes
- Assegurar a execução da campanha atempada e precisa

![My Schedule](images/schedule1.png)

---

### Remarcando uma campanha pendente
Se um utilizador desejar **remarcar uma campanha pendente pré- programada**:
1. Navegar para o **Acção** aba para a campanha desejada.
2. Clique na **Editar** Botão.
3. Selecionar o apropriado **fuso horário** e **novo horário**.
4. Gravar as alterações para confirmar o calendário actualizado.

![Reschedule Campaign](images/schedule2.png)

---

**Dica:** Verifique sempre o fuso horário selecionado para garantir que a campanha envia na hora local pretendida para os destinatários.
