
# Personnes-ressources pour l'exportation

Les **Personnes-ressources pour l'exportation** fonctionnalité dans iTextPro fournit aux utilisateurs un moyen pratique de récupérer les contacts de groupes spécifiques et de les exporter dans différents formats de fichiers (<span data-ph="0"></span> / <span data-ph="1"></span> / <span data-ph="2"></span>). 
La présente section décrit le processus simple d'exportation des contacts.

![Export Contacts](images/export1.png)

---

## Étapes vers l'exportation des contacts

1. **Sélectionner un groupe** 
 Choisissez le groupe spécifique à partir duquel vous voulez exporter des contacts en cliquant sur la case à cocher à côté du nom du groupe.

2. **Lancer l'exportation** 
 Une fois que le groupe souhaité est sélectionné, cliquez sur le **Exportation** option pour lancer le processus d'exportation.

3. **Choisir le format de fichier** 
 Sélectionnez le format de fichier préféré pour les contacts exportés : 
   - <span data-ph="0"></span> 
   - <span data-ph="0"></span> 
   - <span data-ph="0"></span> 

4. **Confirmer et télécharger** 
 Confirmer la sélection d'exportation et procéder au téléchargement du fichier de contacts exportés.

---

## Lignes directrices pour l'utilisation

- **Exportation sélective** – Exportez seulement les groupes dont vous avez besoin pour garder les listes exportées gérables. 
- **Formats multiples** – Choisissez le format de fichier qui correspond le mieux à vos exigences de traitement de données. 
- **Objectif de sauvegarde** – Utilisez cette fonctionnalité pour créer des sauvegardes de listes de contacts importantes.
