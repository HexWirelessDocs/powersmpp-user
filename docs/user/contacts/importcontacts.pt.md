
# Importar Contatos: Streamlined Contato Adição aos Grupos

A **Importar contatos** O recurso no iTextPro facilita um processo simples para os usuários adicionarem contatos aos grupos de forma eficiente. Esta seção delineia o processo simples de três etapas para importar contatos, fornecendo aos usuários várias opções para gerenciamento de contatos simplificado.

![Import Contacts](images/import1.png)

---

## Três Métodos para Importar Contatos

![Import Methods](images/import2.png)

### 1. Importar de um arquivo
Importar contatos em massa de um arquivo (<span data-ph="0"></span> / <span data-ph="1"></span> / <span data-ph="2"></span>). 
Este método é adequado para adicionar um grande número de contatos simultaneamente.

### 2. Copiar e Colar
Copie e cole contatos diretamente de uma planilha Excel. 
Este método oferece uma maneira rápida e flexível de inserir contatos.

### 3. Entrada Manual
Adicionar contatos um por um manualmente. 
Este método é adequado para situações em que é necessário acrescentar um pequeno número de contactos.

![Manual Import](images/import3.png)

---

## Passos para Importar Contatos

1. **Selecionar grupo** 
 Escolha o grupo específico para o qual deseja adicionar contactos no menu suspenso.

2. **Selecionar Método de Importação** 
 Escolha o método preferido para importar contatos: 
   - **Importar de um Ficheiro** – Importação em massa de um arquivo. 
   - **Copiar e Colar** – Copia diretamente e cola contatos do Excel. 
   - **Entrada Manual** – Adicione contatos um por um.

3. **Confirmar e Completar** 
 Confirme a adição de contatos clicando no **Feito** Botão.

---

## Orientações de Utilização

- **Gestão eficiente de grupos** – Use o recurso Importar contatos para gerenciar contatos de forma eficiente dentro de grupos específicos. 
- **Importação em massa** – Para grandes conjuntos de dados, opte pela **Importar de um Ficheiro** método de importação de contactos a granel. 
- **Cópia e Colagem Rápidas** – Use este método para a adição rápida de contatos do Excel. 
- **Entrada Manual** – Ideal para adicionar alguns contatos individualmente. 

!!! note
 Assegurar que o formato do ficheiro (<span data-ph="0"></span>, <span data-ph="1"></span>, <span data-ph="2"></span>) é compatível com **Importar de um Ficheiro** método.
