
# Contactos de exportación

El **Contactos de exportación** característica en iTextPro proporciona a los usuarios una manera conveniente de recuperar contactos de grupos específicos y exportarlos en varios formatos de archivos (<span data-ph="0"></span> / <span data-ph="1"></span> / <span data-ph="2"></span>). 
Esta sección describe el simple proceso de exportación de contactos.

![Export Contacts](images/export1.png)

---

## Pasos para exportar contactos

1. **Seleccionar grupo** 
 Elija el grupo específico del que desea exportar contactos haciendo clic en la casilla de verificación junto al nombre del grupo.

2. **Iniciar la exportación** 
 Una vez seleccionado el grupo deseado, haga clic en **Exportación** opción para iniciar el proceso de exportación.

3. **Elija formato de archivo** 
 Seleccione el formato de archivo preferido para los contactos exportados:

   - <span data-ph="0"></span> 
   - <span data-ph="0"></span> 

4. **Confirmación y descarga** 
 Confirme la selección de exportación y proceda a descargar el archivo de contactos exportados.

---

## Directrices de uso

- **Exportación selectiva** – Exportar sólo los grupos que necesita para mantener las listas exportadas manejables. 
- **Formatos múltiples** – Elija el formato de archivo que mejor se adapte a sus requisitos de procesamiento de datos. 
- **Backup Purpose** – Utilice esta función para crear copias de seguridad de importantes listas de contactos.
