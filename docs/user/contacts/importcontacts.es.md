
# Contactos de importación: Adición de contacto a grupos

El **Contactos de importación** característica en iTextPro facilita un proceso sencillo para que los usuarios añadan contactos a grupos de manera eficiente. Esta sección describe el simple proceso de tres pasos para importar contactos, proporcionando a los usuarios múltiples opciones para la gestión de contactos simplificada.

![Import Contacts](images/import1.png)

---

## Tres métodos para importar contactos

![Import Methods](images/import2.png)

### 1. Importación de un archivo
Contactos de importación a granel de un archivo (<span data-ph="0"></span> / <span data-ph="1"></span> / <span data-ph="2"></span>). 
Este método es adecuado para agregar un gran número de contactos simultáneamente.

### 2. Copiar y Pegar
Copiar y pegar contactos directamente desde una hoja de Excel. 
Este método ofrece una manera rápida y flexible de introducir contactos.

### 3. Entrada manual
Añadir contactos uno a uno manualmente. 
Este método es adecuado para situaciones donde es necesario añadir un pequeño número de contactos.

![Manual Import](images/import3.png)

---

## Pasos a Importar Contactos

1. **Seleccionar grupo** 
 Elija el grupo específico al que desea agregar contactos desde el menú desplegable.

2. **Seleccionar método de importación** 
 Elija el método preferido para importar contactos: 
   - **Importación de un archivo** – Importación a granel de un archivo. 
   - **Copiar y Pegar** – Copiar y pegar directamente contactos de Excel. 
   - **Entrada manual** – Agregar contactos uno por uno.

3. **Confirmación y Completa** 
 Confirme la adición de contactos haciendo clic en el **Hecho** botón.

---

## Directrices de uso

- **Gestión eficiente de grupos** – Utilice la función Contactos de Importación para gestionar eficientemente los contactos dentro de grupos específicos. 
- **Importación a granel** – Para conjuntos de datos grandes, opte por los **Importación de un archivo** método para la importación de contacto masivo. 
- **Copia rápida y sabor** – Utilice este método para la rápida adición de contactos de Excel. 
- **Entrada manual** – Ideal para agregar algunos contactos individualmente. 

!!! note
 Asegúrese de que el formato de archivo (<span data-ph="0"></span>, <span data-ph="1"></span>, <span data-ph="2"></span>) es compatible cuando se utiliza el **Importación de un archivo** método.
