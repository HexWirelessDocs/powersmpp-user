
# Personnes-ressources

Cette section comprend les options suivantes:
- **Gérer le groupe**
- **Groupe d'importation**
- **Groupe d'exportation**

---

## Gestion des paramètres du groupe
Les utilisateurs peuvent accéder à une gamme de paramètres de groupe, y compris la possibilité de :
- Modifier les noms de groupe
- Contacts pour les importations en vrac
- Groupes vides
- Supprimer les groupes

![Manage Group](images/manage1.png)

---

## Options de configuration du groupe

### Modifier le nom du groupe
Les utilisateurs ont la possibilité de modifier le nom d'un groupe spécifique, en assurant la pertinence et la clarté dans l'organisation de contact.

### Contacts pour l'importation en vrac
La fonctionnalité d'importation en vrac permet aux utilisateurs d'ajouter plusieurs contacts à un groupe simultanément, en rationalisant le processus de constitution de groupes de contact.

### Groupe vide
Les utilisateurs peuvent choisir de vider un groupe, en supprimant tous les contacts de ce groupe particulier tout en conservant la structure de groupe pour une utilisation future.

### Supprimer le groupe
L'option de groupe de suppression permet aux utilisateurs de supprimer tout un groupe de contact lorsqu'il n'est plus nécessaire.

---

## Étapes pour gérer les paramètres du groupe

1. **Groupe de gestion de l'accès** 
 Naviguez vers la **Gérer le groupe** section dans la plate-forme iTextPro.

2. **Sélectionner un groupe** 
 Choisissez le groupe spécifique pour lequel vous voulez gérer les paramètres.

3. **Choisir une action de groupe** 
 Selon vos besoins, sélectionnez l'action de groupe appropriée: 
   - **Modifier le nom du groupe** – Modifier le nom du groupe sélectionné. 
   - **Contacts pour l'importation en vrac** – Ajouter plusieurs contacts au groupe sélectionné. 
   - **Groupe vide** – Supprimer tous les contacts du groupe. 
   - **Supprimer le groupe** – Supprimer définitivement tout le groupe.

4. **Enregistrer les modifications** 
 Confirmer et enregistrer les modifications apportées aux paramètres du groupe.
