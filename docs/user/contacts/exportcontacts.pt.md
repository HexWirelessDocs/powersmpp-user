
# Exportar contatos

A **Exportar contatos** recurso no iTextPro fornece aos usuários uma maneira conveniente para recuperar contatos de grupos específicos e exportá-los em vários formatos de arquivo (<span data-ph="0"></span> / <span data-ph="1"></span> / <span data-ph="2"></span>). 
Esta seção descreve o processo simples para exportar contatos.

![Export Contacts](images/export1.png)

---

## Passos para exportar contatos

1. **Selecionar grupo** 
 Escolha o grupo específico do qual deseja exportar contactos clicando na caixa de seleção ao lado do nome do grupo.

2. **Iniciar a Exportação** 
 Uma vez selecionado o grupo desejado, clique no **Exportar** opção de iniciar o processo de exportação.

3. **Escolher formato de arquivo** 
 Selecione o formato de arquivo preferido para os contatos exportados:

   - <span data-ph="0"></span> 
   - <span data-ph="0"></span> 

4. **Confirmar e Transferir** 
 Confirme a seleção de exportação e continue para baixar o arquivo de contatos exportados.

---

## Orientações de Utilização

- **Exportação Seletiva** – Exportar apenas os grupos que você precisa para manter as listas exportadas gerenciáveis. 
- **Vários Formatos** – Escolha o formato de arquivo que melhor se adapta aos seus requisitos de processamento de dados. 
- **Finalidade da Cópia de Segurança** – Use este recurso para criar backups de listas de contatos importantes.
