
# Rapport 4: Télécharger le rapport WhatsApp

Les **Télécharger le rapport WhatsApp** fonction vous permet de télécharger des rapports détaillés de vos campagnes WhatsApp. 
Utilisez ces idées pour analyser les mesures de performance et optimiser vos stratégies futures. 

Cette fonctionnalité garantit également que les **SMS** et **WhatsApp** les campagnes sont maintenues séparément pour une meilleure organisation.
