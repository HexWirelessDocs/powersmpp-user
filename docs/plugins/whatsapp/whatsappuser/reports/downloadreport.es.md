
# Informe 4: Descargar WhatsApp Report

El **Descargar WhatsApp Report** La función le permite descargar informes detallados de sus campañas de WhatsApp. 
Utilice estas ideas para analizar métricas de rendimiento y optimizar sus estrategias futuras. 

Esta funcionalidad también garantiza que los registros para **SMS** y **WhatsApp** las campañas se mantienen separadamente para una mejor organización.
