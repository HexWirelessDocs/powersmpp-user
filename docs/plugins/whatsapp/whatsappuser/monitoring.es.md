
# Supervisión

El **WhatsApp Plugin** ofrece una herramienta de monitoreo dedicada que proporciona información granular sobre la comunicación entre su aplicación y Meta. 
Esta herramienta robusta rastrea cada interacción, desde las solicitudes enviadas a las respuestas recibidas, garantizando la transparencia y la rendición de cuentas.

## Características clave

- **Transaction Records**: Rastrea todas las interacciones entre la aplicación y Meta, registrando cada solicitud y su respuesta correspondiente.
- **Solución de problemas**: Use registros para diagnosticar problemas y solucionar problemas que surjan durante el proceso de comunicación.
- **Período de retención**: Los registros se mantienen para el último **3 días** sólo garantizando información actualizada manteniendo al mismo tiempo eficiente el almacenamiento de datos.

Esta herramienta de monitoreo es crucial para garantizar una comunicación fluida entre la aplicación y Meta, así como para resolver cualquier problema técnico que pueda ocurrir.
