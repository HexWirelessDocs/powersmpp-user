
# Modèle

Un modèle WhatsApp est un format de message préapprouvé fourni par Meta que les entreprises peuvent utiliser pour envoyer des messages automatisés à leurs clients. Ces modèles assurent la conformité aux politiques de messagerie de WhatsApp et aident les entreprises à communiquer de façon cohérente et efficace.

## Pourquoi utiliser des modèles WhatsApp?

- **Conformité:** Les modèles aident les entreprises à respecter les lignes directrices strictes de WhatsApp en matière de messagerie, empêchant ainsi les restrictions de compte. 
- **Efficacité:** Des modèles préapprouvés simplifient le processus de messagerie, économisant temps et efforts. 
- **Cohérence :** Les formats de messagerie cohérents améliorent l'identité de la marque et l'expérience client. 
- **Personnalisation :** Les modèles peuvent être personnalisés avec des variables dynamiques pour livrer des messages personnalisés.

## Catégories de modèles

Les modèles doivent être approuvés dans une catégorie particulière par Meta. Il existe trois catégories:

1. **Commercialisation:** Messages promotionnels envoyés à plusieurs destinataires. 
2. **Utilitaire :** Mises à jour opportunes liées au processus d'achat ou au parcours client. 
3. **Authentification :** Modèles utilisés pour les codes de passe et la vérification de l'utilisateur. 

---

## Guide étape par étape pour ajouter un nouveau modèle

### 1. Sélectionnez une catégorie
Choisissez la catégorie appropriée (Marché, Utilitaire ou Authentification) en fonction de l'objet du message.

### 2. Sélectionnez la langue
Meta prend en charge différentes langues. Sélectionnez la langue dans laquelle le contenu sera livré.

### 3. Ajouter un en-tête (facultatif)
L'en-tête est le titre ou le titre du modèle. Il y a trois options : 
- **Aucun:** Pas de tête. 
- **Texte :** Ajouter un titre de texte. 
- **Médias :** Insérez une image, une vidéo, un document ou un emplacement comme en-tête. 

!!! note
 L'en-tête est optionnel et peut être ignoré si nécessaire.

![Template Header](images/template3.png)

---

### 4. Ajouter le contenu corporel
Le corps est le principal contenu du message qui doit être approuvé par Meta. Les marques peuvent ajouter des variables (placeholders) qui seront remplacées par des valeurs réelles pendant la campagne.

Pour ajouter une variable :
- Cliquez sur **Variable** bouton sous le texte du corps. 
- La variable apparaîtra comme <span data-ph="0"></span>, indiquant la première variable du contenu. 
- Ajouter des variables supplémentaires au besoin (<span data-ph="0"></span>, <span data-ph="1"></span>, etc.).

![Body Content](images/template4.png)

#### Exemple :

Ici, <span data-ph="0"></span> peut être remplacé par le nom du client pendant la campagne et <span data-ph="1"></span> peut être remplacé par le OTP.

Pour le processus d'approbation, vous devez ajouter un exemple dans la case ci-dessous pour chaque variable. Cela aide Meta à comprendre correctement le contenu du message et à approuver le modèle.

![Variable Example](images/template5.png)

### 5. Ajouter un pied de page (facultatif)
Insérer le texte ou les informations de marque comme pied de page si nécessaire. Il s'agit d'un paramètre optionnel et peut être laissé en blanc si nécessaire.

![Footer Example](images/template6.png)

---

### 6. Ajouter des boutons (facultatif)
Les boutons permettent aux destinataires d'agir directement à partir du message. Il y a deux types de boutons :

- **S'écarter :** 
 Exemple : *"Si ce message ne vous intéresse pas, cliquez sur "Opt-Out" pour éviter de recevoir de tels messages à l'avenir."* 
 Vous pouvez également créer une réponse personnalisée en cliquant sur **Bouton personnalisé**.

- **Boutons d'appel à l'action :** 
  - **Visitez le site Web :** Réorienter les utilisateurs vers un site Web (jusqu'à deux boutons autorisés). 
  - **Téléphone :** Laissez les utilisateurs appeler un numéro préconfiguré directement à partir du message.

## Enregistrement et soumission du modèle

Une fois toutes les informations ajoutées, vous pouvez : 
- **Enregistrer en tant qu'ébauche :** Enregistrer le modèle pour une édition ultérieure. 
- **Enregistrer et soumettre pour approbation :** Soumettre le modèle à Meta pour approbation. 

![Save Template](images/template8.png)

Une fois que la marque ajoute le modèle pour approbation, il faut habituellement quelques secondes pour le faire approuver. Une fois approuvée, la marque peut utiliser le modèle pour lancer des campagnes.

![Template Approval](images/template9.png)

---

## Actions post-approbation pour les modèles WhatsApp

Une fois votre modèle WhatsApp approuvé par Meta, vous avez plusieurs options pour le gérer et l'utiliser:

1. **Aperçu :** Consultez le modèle approuvé pour vous assurer qu'il répond à vos attentes. 
2. **Modifier & #160;:** Modifier le contenu du modèle (sous réserve d'une nouvelle approbation par Meta). 
3. **Supprimer:** Supprimer le modèle s'il n'est plus nécessaire. 
4. **Lancement de la campagne :** Lancer directement une nouvelle campagne en utilisant le modèle approuvé. Importez simplement votre liste de contacts cible et envoyez votre message. 

En utilisant efficacement ces actions post-approbation, vous pouvez rationaliser vos efforts marketing WhatsApp et livrer des campagnes d'impact.
