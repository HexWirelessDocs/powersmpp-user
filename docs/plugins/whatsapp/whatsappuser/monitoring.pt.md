
# Acompanhamento

A **Plug- in do WhatsApp** oferece uma ferramenta de monitoramento dedicada que fornece insights granulares sobre a comunicação entre sua aplicação e Meta. 
Essa ferramenta robusta rastreia cada interação, desde solicitações enviadas até respostas recebidas, garantindo transparência e responsabilização.

## Características Principais

- **Registos de transacções**: Acompanha todas as interações entre a aplicação e Meta, registrando cada solicitação e sua resposta correspondente.
- **Resolução de Problemas**: Use logs para diagnosticar problemas e solucionar problemas que surgem durante o processo de comunicação.
- **Período de retenção**: Os logs são retidos para o último **3 dias** Apenas, garantindo informações atualizadas, mantendo o armazenamento de dados eficiente.

Esta ferramenta de monitoramento é crucial para garantir uma comunicação suave entre o aplicativo e Meta, bem como para resolver quaisquer problemas técnicos que possam ocorrer.
