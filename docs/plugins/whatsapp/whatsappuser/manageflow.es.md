
# Manage Flow

Crear y personalizar chatbots inteligentes para automatizar sus interacciones con el cliente y mejorar la eficiencia. Nuestra plataforma ofrece una variedad de plantillas preconstruidas para ayudarle a comenzar rápidamente. Simplemente seleccione una plantilla y adaptarla a sus necesidades específicas, o comience de cero para construir una experiencia única de chatbot.

![Manage Flow Screenshot 1](images/manageflow1.png)

Si prefiere una solución completamente adaptada, haga clic en **Construido desde Scratch** botón para crear un flujo completamente personalizado para su caso de uso único. Esto le redirigirá a una ventana de constructor de flujo donde puede diseñar sus interacciones de chatbot desde el suelo hacia arriba.

## Bot Trigger Condición

Para iniciar el bot, debe definir una condición específica o una palabra clave que activa su activación. Esto puede ser cualquier texto predefinido o frase configurada para iniciar la interacción. 

En caso de que el gatillo inicial no tenga éxito, ofrecemos un **mecanismo de reingreso** para asegurar la activación. Además, puede configurar un **tiempo ocioso**, permitiendo que el bot siga automáticamente con el cliente después de un período de inactividad establecido.

![Bot Trigger Condition](images/manageflow4.png)

## Opciones de flujo

Usted puede elegir entre varias opciones para proceder con el flujo, como **Mensaje**, **Coleccion Input**, **Opción**, **Subdivisión**, y **Enviar adjunto**.

---

### 1. Enviar mensaje

Introduzca el título y escriba el mensaje que desee enviar en el chat.

![Send Message](images/manageflow5.png)

---

### 2. Completo de insumos

Introduzca un mensaje para mostrar al usuario y capturar su entrada. 
La entrada recolectada se puede almacenar en una variable para su uso en la decisión del flujo futuro de la conversación.

![Collect Input](images/manageflow6.png)

---

### 3. Opción

Proveer hasta **3 opciones** para usuarios, según las políticas de Meta. 
El usuario puede seleccionar una opción, y el flujo continuará en consecuencia. También puede subir una imagen que se mostrará antes del mensaje.

![Option](images/manageflow7.png)

---

### 4. Subdivisión

Utilice entradas previamente recogidas para configurar flujos condicionales. 
Dependiendo de las condiciones que se cumplan, el chat procederá de manera diferente.

![Branch Screenshot 1](images/manageflow8.png) 
![Branch Screenshot 2](images/manageflow9.png)

---

### 5. Enviar adjunto

Envíe cualquier tipo de archivo adjunto, como documentos, fotos o videos, a sus usuarios.

![Send Attachment](images/manageflow10.png)
