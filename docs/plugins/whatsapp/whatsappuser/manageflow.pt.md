
# Gerenciar fluxo

Crie e personalize chatbots inteligentes para automatizar as interações do cliente e aumentar a eficiência. Nossa plataforma oferece uma variedade de modelos pré-construídos para ajudá-lo a começar rapidamente. Basta selecionar um modelo e adaptá-lo às suas necessidades específicas, ou começar do zero para construir uma experiência exclusiva de chatbot.

![Manage Flow Screenshot 1](images/manageflow1.png)

Se preferir uma solução completamente personalizada, clique **Compilar a partir de Scratch** botão para criar um fluxo totalmente personalizado para o seu caso de uso exclusivo. Isso irá redirecioná-lo para uma janela de builder de fluxo onde você pode projetar suas interações chatbot do zero para cima.

![Manage Flow Screenshot 2](images/manageflow2.png)

Aqui está um exemplo de como um fluxo criado pode parecer:

## Condição do gatilho de Bots

Para iniciar o bot, você deve definir uma condição específica ou palavra-chave que desencadeia sua ativação. Este pode ser qualquer texto ou frase pré-definida configurada para iniciar a interação. 

No caso de o gatilho inicial não ser bem sucedido, oferecemos um **mecanismo de repetição** para garantir a ativação. Além disso, você pode configurar um **tempo- limite inativo**, permitindo que o bot para automaticamente acompanhar com o cliente após um período definido de inatividade.

![Bot Trigger Condition](images/manageflow4.png)

## Opções de Fluxo

Você pode escolher entre várias opções para prosseguir com o fluxo, como **Enviar mensagem**, **Recolha a Entrada**, **Opção**, **Ramo**, e **Enviar Anexo**.

---

### 1. Enviar mensagem

Digite o título e escreva a mensagem que deseja enviar no chat.

![Send Message](images/manageflow5.png)

---

### 2. Recolha a entrada

Digite uma mensagem para mostrar ao usuário e capture suas entradas. 
A entrada coletada pode ser armazenada em uma variável para uso na decisão do fluxo futuro da conversa.

![Collect Input](images/manageflow6.png)

---

### 3. Opção

Fornecer até **3 opções** para usuários, conforme as políticas da Meta. 
O usuário pode selecionar uma opção, e o fluxo continuará de acordo. Você também pode enviar uma imagem para ser exibida antes da mensagem.

![Option](images/manageflow7.png)

---

### 4. Ramo

Use entradas coletadas anteriormente para configurar fluxos condicionais. 
Dependendo de quais condições são cumpridas, o chat irá proceder de forma diferente.

![Branch Screenshot 1](images/manageflow8.png) 
![Branch Screenshot 2](images/manageflow9.png)

---

### 5. Enviar Anexo

Envie qualquer tipo de anexo, como documentos, fotos ou vídeos, para seus usuários.

![Send Attachment](images/manageflow10.png)
