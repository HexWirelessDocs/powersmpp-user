
# Gérer le flux

Créez et personnalisez des chatbots intelligents pour automatiser vos interactions client et améliorer l'efficacité. Notre plateforme offre une variété de modèles préconstruits pour vous aider à démarrer rapidement. Il suffit de sélectionner un modèle et de l'adapter à vos besoins spécifiques, ou de commencer à zéro pour construire une expérience de chatbot unique.

![Manage Flow Screenshot 1](images/manageflow1.png)

Si vous préférez une solution entièrement adaptée, cliquez sur **Construire à partir de Scratch** bouton pour créer un flux entièrement personnalisé pour votre cas d'utilisation unique. Cela vous redirigera vers une fenêtre de construction de flux où vous pouvez concevoir vos interactions chatbot à partir du sol.

![Manage Flow Screenshot 2](images/manageflow2.png)

Voici un exemple de ce à quoi un flux créé peut ressembler:

![Manage Flow Screenshot 3](images/manageflow3.png)

## Condition de déclenchement du bot

Pour lancer le bot, vous devez définir une condition ou un mot clé spécifique qui déclenche son activation. Cela peut être n'importe quel texte ou phrase prédéfini configuré pour démarrer l'interaction. 

Si le déclencheur initial est infructueux, nous offrons **mécanisme de réessayer** pour assurer l'activation. En outre, vous pouvez configurer un **temps mort**, permettant au bot de suivre automatiquement le client après une période d'inactivité définie.

![Bot Trigger Condition](images/manageflow4.png)

## Options de débit

Vous pouvez choisir parmi différentes options pour procéder avec le flux, comme **Envoyer un message**, **Collecte des données**, **Option**, **Secteur**et **Envoyer la pièce jointe**.

---

### 1. Envoyer un message

Entrez le titre et écrivez le message que vous voulez envoyer dans le chat.

![Send Message](images/manageflow5.png)

---

### 2. Recueillir les données

Saisissez un message à afficher à l'utilisateur et capturez leur entrée. 
L'entrée collectée peut être stockée dans une variable pour être utilisée pour décider du flux futur de la conversation.

![Collect Input](images/manageflow6.png)

---

### 3. Option

Fournir jusqu'à **3 options** pour les utilisateurs, selon les politiques de Meta. 
L'utilisateur peut sélectionner une option, et le flux continuera en conséquence. Vous pouvez également télécharger une image à afficher avant le message.

![Option](images/manageflow7.png)

---

### 4. Service

Utiliser les entrées précédemment collectées pour configurer les flux conditionnels. 
Selon les conditions qui sont remplies, la conversation se déroulera différemment.

![Branch Screenshot 1](images/manageflow8.png) 
![Branch Screenshot 2](images/manageflow9.png)

---

### 5. Envoyer la pièce jointe

Envoyez tout type de pièce jointe, comme des documents, des photos ou des vidéos, à vos utilisateurs.

![Send Attachment](images/manageflow10.png)
