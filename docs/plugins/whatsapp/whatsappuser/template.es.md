
# Plantilla

Una plantilla de WhatsApp es un formato de mensaje preaprobado proporcionado por Meta que las empresas pueden utilizar para enviar mensajes automatizados a sus clientes. Estas plantillas aseguran el cumplimiento de las políticas de mensajería de WhatsApp y ayudan a las empresas a ofrecer una comunicación coherente y eficaz.

## ¿Por qué utilizar las plantillas de WhatsApp?

- **Cumplimiento:** Las plantillas ayudan a las empresas a cumplir con las estrictas directrices de mensajería de WhatsApp, evitando las restricciones de la cuenta. 
- **Eficiencia:** Las plantillas aprobadas previamente simplifican el proceso de mensajería, ahorrando tiempo y esfuerzo. 
- **Consistencia:** Los formatos de mensajería consistentes mejoran la identidad de marca y la experiencia del cliente. 
- **Personalización:** Las plantillas se pueden personalizar con variables dinámicas para ofrecer mensajes personalizados.

## Plantilla Categorías

Las plantillas deben ser aprobadas bajo una categoría específica por Meta. Hay tres categorías:

1. **Marketing:** Mensajes promocionales enviados a múltiples destinatarios. 
2. **Utilidad:** Actualizaciones oportunas relacionadas con el proceso de compra o viaje al cliente. 
3. **Autenticación:** Plantillas utilizadas para los códigos de paso únicos y la verificación del usuario. 

---

## Guía paso a paso para agregar una nueva plantilla

### 1. Seleccione una categoría
Elija la categoría apropiada (Marketing, Utilidad o Authentication) basado en el propósito del mensaje.

### 2. Seleccionar idioma
Meta admite varios idiomas. Seleccione el idioma en el que se entregará el contenido.

### 3. Añadir un encabezado (opcional)
El encabezado es el título o encabezado de la plantilla. Hay tres opciones: 
- **Ninguno.** Sin cabecera. 
- **Texto:** Añadir un título de texto. 
- **Medios:** Insertar una imagen, vídeo, documento o ubicación como encabezado. 

!!! note
 El encabezado es opcional y puede ser saltado si no es necesario.

![Template Header](images/template3.png)

---

### 4. Agregar el contenido del cuerpo
El cuerpo es el contenido principal del mensaje que debe ser aprobado por Meta. Las marcas pueden añadir variables (pantallas) que serán reemplazadas por valores reales durante la campaña.

Para añadir una variable:
- Haga clic en **Variable** botón debajo del texto del cuerpo. 
- La variable aparecerá como <span data-ph="0"></span>, indicando la primera variable en el contenido. 
- Añadir variables adicionales según sea necesario (<span data-ph="0"></span>, <span data-ph="1"></span>, etc.).

![Body Content](images/template4.png)

#### Ejemplo:

Aquí, <span data-ph="0"></span> puede ser reemplazado por el nombre del cliente durante la campaña y <span data-ph="1"></span> puede ser reemplazado por el OTP.

Para el proceso de aprobación, es necesario añadir un ejemplo en el cuadro que se proporciona a continuación contra cada variable. Esto ayuda a Meta a entender el contenido del mensaje correctamente y aprobar la plantilla.

![Variable Example](images/template5.png)

---

### 5. Agregue un pie (Opcional)
Inserte la información de texto o marca como el pie de página si es necesario. Es un parámetro opcional y se puede dejar en blanco si no es necesario.

![Footer Example](images/template6.png)

---

### 6. Add Buttons (Optional)
Los botones permiten a los receptores tomar acción directamente desde el mensaje. Hay dos tipos de botones:

- **Opt-Out:** 
 Ejemplo: *"Si no estás interesado en este mensaje, haz clic en 'Opt-Out' para evitar recibir dichos mensajes en el futuro."* 
 También puede crear una respuesta personalizada haciendo clic en **Botón personalizado**.

- **Botones Call-to-Action (CTA):** 
  - **Visit Website:** Redirección de usuarios a un sitio web (hasta dos botones permitidos). 
  - **Número de teléfono:** Deje que los usuarios llamen a un número preconfigurado directamente desde el mensaje.

## Ahorro y presentación de la plantilla

Una vez que se añada toda la información, puede: 
- **Guardar como proyecto:** Guardar la plantilla para la edición posterior. 
- **Guardar y enviar para su aprobación:** Enviar la plantilla a Meta para su aprobación. 

![Save Template](images/template8.png)

Una vez que la marca agrega la plantilla para su aprobación, generalmente toma unos segundos para que sea aprobado. Una vez aprobado, la marca puede utilizar la plantilla para iniciar campañas.

![Template Approval](images/template9.png)

---

## Acciones posteriores a la aprobación para las plantillas de WhatsApp

Una vez que su plantilla WhatsApp es aprobada por Meta, tiene varias opciones para gestionar y utilizarla:

1. **Vista previa:** Vea la plantilla aprobada para asegurar que cumple sus expectativas. 
2. **Editar:** Modificar el contenido de la plantilla (sujeto a la nueva aprobación por Meta). 
3. **Suprímase:** Retire la plantilla si ya no es necesaria. 
4. **Iniciar la campaña:** Inicia directamente una nueva campaña usando la plantilla aprobada. Simplemente importa tu lista de contactos objetivo y envía tu mensaje. 

Mediante la utilización eficaz de estas acciones posteriores a la aprobación, puede simplificar sus esfuerzos de marketing de WhatsApp y realizar campañas impactantes.
