
# Gérer efficacement votre équipe avec les comptes mandataires

Les **Gérer les agents** section vous permet de créer des comptes individuels pour les membres de votre équipe, permettant un accès sécurisé aux conversations de chat web. En attribuant des agents spécifiques à différents flux de webchat, vous pouvez optimiser l'efficacité et la responsabilité de votre équipe.

## Principaux avantages de l'utilisation des comptes des mandataires

- **Gestion d'équipe améliorée**: Organisez votre équipe en groupes spécialisés pour gérer différents sujets de webchat ou segments clients.
- **Amélioration de la sécurité**: Protéger les informations sensibles des clients en limitant l'accès aux agents autorisés.
- **Responsabilité accrue**: Surveiller les performances de chaque agent et identifier les domaines à améliorer.
- **Programme de poste flexible**: Créer des comptes d'agents séparés pour différents déplacements afin d'assurer une couverture transparente.

## Ajout d'un nouvel agent

Pour ajouter un nouvel agent :

1. Cliquez sur **Ajouter un nouvel agent**.
2. Remplissez les détails requis.
3. Enregistrez les changements.

Une fois enregistré, le nouveau compte d'agent sera créé instantanément, ce qui permettra à votre équipe de gérer efficacement les interactions de webchat.
