# Bem-vindo ao Guia de Utilizador SMPP Power

O Power SMPP é um gateway de mensagens de alto desempenho e confiável projetado para operadores de telecomunicações, agregadores e empresas.

Esta documentação abrange os recursos voltados para o usuário, incluindo mensagens, contatos, relatórios e APIs de desenvolvedores.

Para mais informações, visite o site oficial: [Power SMPP.com](https://www.powersmpp.com){ target="_blank" }
