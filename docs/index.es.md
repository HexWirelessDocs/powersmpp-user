# Bienvenido a Power SMPP User Guide

Power SMPP es un portal de mensajería de alto rendimiento y fiable diseñado para operadores de telecomunicaciones, agregadores y empresas.

Esta documentación cubre las funciones de usuario que incluyen mensajería, contactos, informes y API de desarrolladores.

Para más información, visite el sitio web oficial: [Power SMPP.com](https://www.powersmpp.com){ target="_blank" }
